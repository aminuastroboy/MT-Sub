# MT-Sub Frontend

Vercel-ready Next.js frontend demo for Musatech Solutions Ltd.

## Features included
- Public homepage
- About, pricing and contact pages
- Login and register pages
- User dashboard
- Buy airtime page
- Buy data page
- Wallet page
- Transactions page
- Profile page
- Admin dashboard
- Admin users, transactions, networks, plans, pricing and settings pages

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this folder to a GitHub repository.
2. Import the repository into Vercel.
3. Use the default Next.js settings.
4. Deploy.

## Notes
This is a frontend demo only. Purchase actions and wallet funding are currently UI placeholders and should be connected to a backend/API next.
